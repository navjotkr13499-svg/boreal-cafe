##  About The Project

**Boreal Café** is a concept commercial website designed for a modern café brand.
This project was built as part of my personal portfolio to showcase my ability to 
create engaging, interactive, and visually rich web experiences using core web 
technologies.

The website features:
-  Smooth **3D animations** for an immersive user experience
-  Modern and clean **UI/UX design**
-  Designed with a **creative visual layout**
-  Built entirely with **pure HTML** — no frameworks needed
